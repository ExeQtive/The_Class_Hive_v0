# TheClassHive

A unified workspace for K-12 teachers. Built with Next.js, Supabase, and Tailwind CSS.

Live at [www.theclasshive.com](https://www.theclasshive.com)

## Stack
- Next.js 16
- Supabase (Postgres + RLS)
- Tailwind CSS + shadcn/ui
- Vercel
